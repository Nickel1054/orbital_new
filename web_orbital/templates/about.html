About


This site contains an application of building a visualisation of orbits, depending on the object. There are orbital parameters written in database for each planet of Solar System.

If you want to add an existing object to visualisation, use the Add existing body button.

If you want to add a self-specified object to visualisation, use the Add custom body button.

Be warned: for comets and asteroids with high eccentricity plots may become distorted. This is still to be fixed.

For any bugs and problem reports feel free to contact us.
